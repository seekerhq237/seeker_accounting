1- LES DIFFERENTES PROCEDURES DE SOUMISSION / FLILING OPTIONS 
 
Il y a 3 façons simples de soumettre votre DSF :  
 There exist three options 
1- Par saisie directe dans l’interface (non recommandée) ; Manually on the interface 
2- Par télé-versement du fichier Excel (Format DGI) pour les systèmes comptables « non web 
based »;  
Uploading the approved DGT Excel STR format for non-web based accounting systems; 
3- Par Interface de programmation (API) - si vous avez un système comptable web-based .  
Through the Application Programming Interface (API) - for web based accounting systems 
(after having updated your API system, Note 3F has been modified).

Click on the « DSF (télédéclaration) » link.

Follow the instructions to obtain an authentication token, then create, add and submit all 
pages of a return from your online accounting system.

5- CHOIX DE LA DEVISE ET DU TAUX DE CHANGE / CHOOSING A CURRENCY 
AND EXCHANGE RATE 
 
Spécifiez la devise et le taux de change si vous payez votre solde 
en devise (EURO / DOLLAR) puis valider en cliquant sur le bouton 
« Enregistrer ». 
Specify the currency and the exchange rate if you pay your 
balance in currency (EURO / DOLLAR) then confirm by clicking on 
the "Save" button. 
Par défaut le système utilise XAF (franc CFA de l’Afrique Centrale). / By default the 
system uses XAF (Central African Franc CFA).
